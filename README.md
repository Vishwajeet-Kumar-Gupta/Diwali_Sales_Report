# Diwali_Sales_Report
# 🎯 Diwali Sales Data Analysis Project

## 📌 Project Overview

This project analyzes a **Diwali Sales dataset containing 11,000+ records** to uncover customer purchasing behavior, sales trends, and business insights. The analysis helps businesses make data-driven decisions for marketing and sales optimization.

---

## 📊 Dataset Details

* 📁 Dataset Size: 11,000+ records
* 📌 Features:

  * Customer ID
  * Gender
  * Age Group
  * State
  * Occupation
  * Product Category
  * Orders
  * Amount (Sales)

---

## 🛠️ Tools & Technologies Used

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Jupyter Notebook
* Data Cleaning & Exploratory Data Analysis (EDA)

---

## 🔍 Key Analysis Performed

* Data Cleaning (handling missing & duplicate values)
* Exploratory Data Analysis (EDA)
* Customer segmentation based on gender, age, and state
* Sales trend analysis
* Product category performance

---

## 📈 Key Insights

* Female customers contributed more to overall sales
* Age group **26–35** showed highest purchasing power
* Top-performing states generated maximum revenue
* Certain product categories dominated sales

---

## 🚀 Conclusion

The analysis highlights key customer segments and high-performing products, helping businesses improve **target marketing strategies and increase revenue**.
